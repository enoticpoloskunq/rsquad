# 🦝 Raccoon Squad

Android VPN client with proper VLESS URI parsing and fragment support.

## Features
- ✅ Correct VLESS URI parsing (fragment, noise, mtu actually work!)
- ✅ Xray-core based
- ✅ Real-time connection logs
- ✅ Modern UI (Jetpack Compose)

## Building
Requires Android Studio or build via GitHub Actions.
